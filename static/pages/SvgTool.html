Career Pathways SVG Helper

This tool is designed to help link the SVG charts with the pages that they represent. It takes SVG input and provide new SVG Output.

For those who are more technical:

To use, simple follow these steps:

  1. Copy and paste the svg code into the input below
  2. Press "Proces" and copy the output back into the SVG file
  3. Save and view in browser, See if any are red If so, see step 8 of the process below.

Easy step by step guide:

To use, simple follow these steps:

  1. Open the SVG file in Notepad
    • Open Notepad
    • Click File -> Open
    • Change filetype to "All Files (*.*)"
    • Browse to SVG file and Open
    • Keep the notepad open for later use.
  2. Copy all the SVG code
  3. Paste all SVG code in the Input below
  4. Ensure the correct pathway for the SVG is selected from the drop-down list
  5. Press the "Process" button
  6. Copy the new SVG code from the output
  7. Paste the code back into the SVG file open in Notepad and Save
  8. Open the SVG in a browser to display the new SVG.
  9. If any of the circles are red, it failed to find a link for that job. To resolve:
    • Check that the job exists in the sharepoint list, if not then create.
    • Ensure the item in the list doesnt end in a space.
    • Ensure the correct pathway has been selected
    • Press the process button again.
    • Repeat point 5 onwards
  10. When ready, upload to sharepoint
